# Frontend Web Development

A cumulative set of exercises for the Web-sovellusten perusteet (Web Application Basics) course at OAMK.
Each task builds on the previous one, forming a single evolving web application that grows throughout the course.

## Course Goals
- Build web pages using HTML and CSS
- Create browser‑based programs using JavaScript following object‑oriented principles
- Implement asynchronous requests from the browser (HTTP/HTTPS)

## Course Content
- HTML & CSS programming
- JavaScript fundamentals
- JavaScript object‑oriented programming
- Browser‑based JavaScript
- Asynchronous communication with APIs

## Weekly Topics
- **HTM**L – structure and semantic markup
- **CSS** – layout, styling, responsive design
- **Visual** design – colors, typography, UI basics
- **JavaScript basics** – variables, functions, events
- **JavaScript language features** – arrays, objects, loops
- **JavaScript OOP** – classes, methods, encapsulation
- **HTTP(S) with JavaScript** – fetch, async/await, API calls

## Assessment & Workflow
- Weekly assignments submitted through GitHub Classroom
- Optional personal web application project

## Purpose of This Repository
This repository documents the full progression of the course through a single, evolving web application.
It demonstrates growth in HTML, CSS, JavaScript, OOP, and asynchronous browser programming.
